# Source Awakenings starter site

Repository: Cre8torIAM/sourceawakenings

## Upload
Upload every file and the `assets` folder to the root of the GitHub repository.

## Publish with GitHub Pages
Repository Settings > Pages > Deploy from a branch > main > /(root) > Save.

## Main navigation
- Home
- Reflections
- The Journey
- Books
- About & Connect

The privacy page is linked only in the footer and does not need to appear in the main navigation.

## Important
Do not change domain DNS records until the temporary GitHub Pages version is working. Preserve all Website.com MX/TXT/CNAME email records when the domain is connected later.
